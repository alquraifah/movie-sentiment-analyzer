Overview

What is Sentiment Analysis?

Sentiment analysis is an NLP technique that detects the emotional tone behind text — helping machines understand human opinion at scale.

Text Preprocessing

HTML removal, punctuation stripping, lowercasing, stopword removal, and Porter stemming transform raw reviews into clean tokens.

Feature Extraction

TF-IDF and Bag-of-Words vectorizers convert text into numerical feature matrices with 5,000 features each, ready for ML input.

ML Classification

Naive Bayes, Logistic Regression, and Neural Network models are trained and compared. The best model is deployed for live prediction.

Pipeline

End-to-End ML Pipeline

1
Load & Inspect

Read IMDB Dataset.csv, remove duplicates and null values.

2
Preprocess Text

HTML → symbols → lowercase → stopwords → stemming.

3
Encode Labels

positive → 1  /  negative → 0

4
Train / Test Split

80% training  /  20% testing (random_state=42).

5
Vectorize

TF-IDF and BoW, max_features=5000.

6
Train & Evaluate

NB, LR, MLP — accuracy, precision, recall, F1.

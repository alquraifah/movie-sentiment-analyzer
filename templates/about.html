Project

About This Project

A complete end-to-end NLP pipeline for binary sentiment classification of IMDB movie reviews — from raw text to live web predictions.

Dataset

IMDB Movie Reviews — 50,000 labelled reviews (25K positive, 25K negative). Sourced from the Stanford AI Lab's Large Movie Review Dataset.

Tech Stack

Python · Flask · scikit-learn · NLTK · BeautifulSoup4 · pandas · Bootstrap 5 · Chart.js

Goal

Binary classification: Positive vs Negative sentiment — evaluated by Accuracy, Precision, Recall and F1-Score.

Full ML Pipeline
1
Import Libraries

pandas, numpy, sklearn, nltk, BeautifulSoup — all standard Python scientific libraries for NLP and ML.

2
Load & Inspect IMDB Dataset

Read IMDB Dataset.csv with pandas. Inspect shape, column types, and sentiment distribution. Drop duplicates and nulls.

3
Text Preprocessing

Five sequential steps applied to every review:

  1. Remove HTML tags with BeautifulSoup
  2. Remove symbols and punctuation ([^a-zA-Z])
  3. Convert to lowercase
  4. Remove English stop-words (NLTK)
  5. Apply Porter Stemmer
4
Encode Sentiment Labels

Map positive → 1 and negative → 0 using pandas .map().

5
Train / Test Split

80/20 stratified split with random_state=42. 39,665 training reviews, 9,917 test reviews.

6
Feature Extraction

Both TF-IDF and Bag of Words vectorizers, each limited to the top 5,000 features. Vectorizers are fit on training data only to prevent data leakage.

7
Train Models

Five model/vectorizer combinations trained and saved: Naive Bayes ×2, Logistic Regression ×2, Neural Network ×1.

8
Evaluate & Compare

Accuracy, Precision, Recall, F1-Score and Confusion Matrix computed for all models. Results saved to models/metrics.json.

Why These Models?
Naive Bayes

Probabilistic baseline — fast to train, requires little data, and works naturally with word count features (BoW). Assumes word independence, which is often good enough for sentiment.

Logistic Regression

Strong linear classifier for high-dimensional sparse text data. Pairs excellently with TF-IDF. Provides probability outputs for confidence scores. Interpretable and computationally efficient.

Neural Network (MLP)

Non-linear model capable of capturing complex patterns. Uses a single hidden layer of 100 neurons as a representative deep learning comparison. Slightly slower but competitive accuracy.
